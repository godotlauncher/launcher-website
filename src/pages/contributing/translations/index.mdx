# Help Translate Godot Launcher

Godot Launcher includes localized interface strings. Translation contributions help keep the app clear, accurate, and natural for more users.

## Quick ways to help

- **Try the launcher in your language.** Switch to your locale in `Settings > Appearance > Language` and note any phrases that feel off.
- **Report wording issues.** Small corrections are good candidates for a GitHub issue or a Discord message. You do not need to open a pull request for every typo.
- **Review existing translations.** If you know a supported language well, review interface text for accuracy, tone, and missing context.

If you only want to report a typo or awkward phrase, open a [GitHub issue](https://github.com/godotlauncher/launcher/issues/new/choose) or leave a note in the [community hub](/community/).

## Full checklist

For translation pull requests, use the full contribution checklist:

- [Translation contribution docs](https://docs.godotlauncher.org/contributing/translations/)
- [CONTRIBUTING_TRANSLATIONS.md](https://github.com/godotlauncher/launcher/blob/main/CONTRIBUTING_TRANSLATIONS.md)

The checklist covers file locations, JSON rules, placeholders, validation tests, and pull request expectations.

## Coordinate

- Coordinate with other translators or reviewers on [Discord](https://discord.gg/Ju9jkFJGvz), especially when a phrase depends on Godot terminology or platform behavior.
- Visit the main [Contribute page](/contribute/) for broader project roles.
- Use the docs checklist when you are ready to open a pull request.
